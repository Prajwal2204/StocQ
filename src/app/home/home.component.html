What is DQN?

This application uses a Model-free Reinforcement Learning technique called Deep Q-Learning. At any given time (episode), an agent observes its current state, selects and performs an action (buy/sell/hold), observes a subsequent state, receives some reward signal and lastly adjusts it's parameters based on the gradient of the loss computed

A DQN, or Deep Q-Network, basically approximates a state-value function in a Q-Learning framework with a neural network which we exploit for prediction purposes.

Our Main Objectives

  • Build a sustainable and fast model.
  • Implement a risk management strategy for the model.
  • Build an analytics tool to help people judge the results through a long term view.
  • Compare and test results to real life scenarios.
  • Connect to the real-time market for paper trade execution, if the model has turned out as we expected
